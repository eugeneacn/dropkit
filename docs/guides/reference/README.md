# Reference

Exhaustive, information-oriented documentation. For the precise
definition of every flag, every metric, every exit code, every output
field.

_Not yet written._ The authoritative source today is the spec set in
[`docs/specs/`](../../specs/):

- [`flow-metrics.md`](../../specs/flow-metrics.md) — flags, metric
  definitions, output schema, error codes, edge cases.
- [`ai-adoption-report.md`](../../specs/ai-adoption-report.md) — flags,
  delta math, output schema, mode-specific rules.

Specs are normative; if a future reference page disagrees with a spec,
the spec wins.
